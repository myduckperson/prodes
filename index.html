Проте завдяки важкому навчанню та праці цнй проєкт став можливий, тому я дуже вдячний

МОЇ СПРОБИ В АДАПТИВНЕ МЕНЮ ДЛЯ МОБІЛЬНИХ ПРИСТРОЇВ

Для опанування простих знань CSS та HTML витрачається куди менше часу, ніж на опанування JavaScript та JQuery.
Тому на перенесене на останні дні адаптивне меню для мобільних пристроїв невистачило часу, проте потрібні були додатеові лічені години для виконання цього завдання

ПРОЦЕСС РОЗРОБКИ

Цей проєкт потребував великої кількості часу на виконання та обмеженої кількості днів,
ще й на додачу пропущений час на початку, що змусило навчатися та працювати над завданням цілими днями.
Веб-дизайн й без цього рутинна праця, а з перелічними вище факторами працювати було б майже тошно, якби
не мій креативний підхід до праці, по-перше музичне супроводження праці, що не є дивним
по-друге текстове сприйняття інформації, що допомогло уникнути вимикання музики та підтримки темпу навчання (проте було й складніше розуміти)
по-третє залучення для прототипів картинок та іншіх об'єктів речей, які для тебе цікаві, розважаючі та захоплюючі (в моєму випадку серія игор дум) знімая стресс рутинної праці
та додає незвичності
